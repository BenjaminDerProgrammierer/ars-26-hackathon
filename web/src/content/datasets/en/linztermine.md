The festival-window query reviewed on July 13 returned 187 city events. Event
occurrences link to companion location, organizer, and tag registries; location
ID 358 and organizer ID 7 are reviewed Ars Electronica Center anchors. The live
XML is German-only, has no event coordinates, and needs explicit date parameters,
encoding-aware parsing, and a cached fallback before teams use it.

Companion sources: [locations](https://www.data.gv.at/katalog/datasets/3cca23c2-2aa6-4421-96db-8b914de62d56),
[organizers](https://www.data.gv.at/katalog/datasets/9c0a65e3-db8c-4784-98df-b856a9cd3576),
and [tags](https://www.data.gv.at/katalog/datasets/fb08a46d-69ca-4fb0-baac-67557c933341).
