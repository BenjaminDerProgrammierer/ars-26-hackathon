## Beschreibung

Die Standortdatei umfasst rund 134 Punkte, die Nutzungsdatei rund 107 Einträge für Jänner bis Juli 2022. WGS84-Koordinaten verwenden quoted Dezimalkomma-Werte; Standort- und Nutzungsnamen benötigen vor einer Verknüpfung Normalisierung. Die Daten erlauben keine Aussage über Verfügbarkeit oder Leistung im Jahr 2026.
