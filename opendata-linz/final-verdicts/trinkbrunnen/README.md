# Trinkbrunnen

**Final verdict: USE WITH PREPARATION.** Correct and verify the CRS, then include
only potable/in-service fountains in the visitor-services bundle.

**Rainer verdict:** ✅ Recommended — genuinely useful drinking-fountain locations
for an outdoor September festival, with one solvable coordinate-CRS gotcha.

The current CSV includes 132 fountain records; 86 are drinking fountains and 13
ornamental fountains are marked suitable for drinking. Operating months cover the
September festival.

Metadata incorrectly describes projected meter-like coordinates as EPSG:4326.
Confirm the actual CRS and axis order with the publisher before conversion; see
[issue #7](https://github.com/BenjaminDerProgrammierer/ars-26-hackathon/issues/7).
Do not treat water status or opening hours as current without verification.
