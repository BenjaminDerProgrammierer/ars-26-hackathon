# Linztermine - VeranstalterInnen

**Final verdict: USE.** Bundle it with Linztermine events and locations.

**Rainer verdict:** ✅ Recommended — live, well-populated organizer registry that
includes Ars Electronica (ID 7) and other festival venues; text/address join.

The live XML contains about 1,611 organizers with stable IDs, names, addresses,
phone numbers, and websites. Organizer ID 7 is Ars Electronica Linz GmbH & Co KG.

Convert XML to UTF-8 JSON and publish a small curated organizer match table. It is
not independently geospatial and only a subset is festival-relevant.
