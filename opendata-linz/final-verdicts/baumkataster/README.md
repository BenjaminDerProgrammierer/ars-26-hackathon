# Baumkataster

**Final verdict: USE.** Include it in the default bundle. This is the strongest
single Linz dataset for the hackathon.

**Rainer verdict:** ✅ Recommended — current, clean, geocoded tree inventory that
joins directly to venue coordinates.

The July 2026 export contains about 27,000 trees with WGS84 coordinates, species,
height, crown diameter, and trunk circumference. It supports green-route scoring,
tree exploration, venue comparisons, and estimated shade/canopy visualizations.

Preparation is light: parse the semicolon CSV, normalize missing-value markers,
and correct invalid festival venue coordinates before proximity joins. Do not
describe crown diameter as measured shade or thermal comfort.
