# Behindertenstellplätze

**Final verdict: USE WITH PREPARATION.** Include only as a converted, dated
prototype layer.

**Rainer verdict:** ⚠️ Borderline — real, complete accessible-parking point data,
but shipped only as a Shapefile in a projected CRS, so it needs conversion and
reprojection before use.

The 574 points can support nearest accessible parking and same-street matches to
festival venues.

Convert the complete Shapefile from MGI Austria GK Central to WGS84, decode legacy
DBF text, and retain addresses. The 2022 vintage must be visible; do not present it
as an authoritative guarantee of current accessible parking.
