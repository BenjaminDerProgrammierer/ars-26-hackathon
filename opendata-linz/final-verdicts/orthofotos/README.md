# Orthofotos

**Final verdict: OPTIONAL SHOWCASE SOURCE.** Offer only preselected, web-ready
venue tiles to teams that explicitly want image/time-travel work.

**Rainer verdict:** ⚠️ Borderline — visually compelling time-travel idea, but
heavy TIFF-only delivery in a non-WGS84 projection is real hackathon friction.

The 1988-2021 vintages support aerial then-and-now sliders. The 2019 vintage alone
has about 504 tiles at roughly 23 MB each, around 11.8 GB total.

Use the overview geometry to select festival-area tiles, reproject EPSG:31255,
and publish COG/PNG/web tiles. Do not ask general participants to download or
convert a complete vintage onsite.
