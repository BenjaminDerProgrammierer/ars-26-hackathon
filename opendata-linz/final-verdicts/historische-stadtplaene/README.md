# Historische Stadtpläne

**Final verdict: USE WITH PREPARATION.** Pre-tile/downsample a small set of maps
covering festival venues; do not make teams process the source rasters onsite.

**Rainer verdict:** ✅ Recommended — georeferenced historical raster maps in
WGS84, drop-in for "then & now" venue overlays.

The 1876-1990 series is well suited to venue history, map sliders, and narrative
walks. EPSG:4326 and EPSG:3857 variants avoid reprojection for web maps.

Full-resolution TIFF/JPG assets are large and each year is a separate record.
Prepare browser tiles and record each source vintage. This is a spatial backdrop,
not an attribute dataset.
