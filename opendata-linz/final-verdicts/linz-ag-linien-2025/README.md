# LINZ AG Linien 2025

**Final verdict: USE WITH PREPARATION.** Publish a WGS84 GeoJSON snapshot in the
default bundle; retain GML/GeoPackage for advanced teams.

**Rainer verdict:** ✅ Recommended — clean WGS84 stop/route geodata; the caveat is
it is static infrastructure, not a live timetable.

The source has 748 stops plus route lines and a routable graph. It supports nearest
stop, route overlays, and the static half of event-to-event mobility planning.

Convert GML/GeoPackage to GeoJSON and split comma-joined line identifiers. Do not
promise schedules or arrival feasibility without the separate EFA service.
