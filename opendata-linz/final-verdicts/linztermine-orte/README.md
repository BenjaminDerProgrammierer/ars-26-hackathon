# Linztermine - Orte

**Final verdict: USE.** Bundle it with Linztermine events as the venue identity
bridge.

**Rainer verdict:** ✅ Recommended — clean venue registry with a direct,
name-based join to Ars venues (location 358 = Ars Electronica Center confirmed).

The live XML contains about 428 locations/sites with IDs, parent relationships,
names, addresses, and websites. Ars Electronica Center is ID 358; Main Deck and
Deep Space are child sites. This mirrors the festival's building/room hierarchy.

Normalize whitespace and names, then create an explicit match table to cleaned
festival locations. There are no coordinates in this source, so it is not a
standalone spatial dataset.
