# Beherbergungsbetriebe

**Final verdict: DO NOT USE for the proposed hotel/venue use case.**

**Rainer verdict:** ❌ Drop — the name is misleading; this is city-wide monthly
tourism data, not a list of accommodation establishments, and it has no locations.

The files contain monthly arrivals and overnight stays split into domestic and
foreign totals. There are no hotel names, addresses, rooms, or coordinates, and
the newest year is 2024.

It could supply one historical September context number, but that does not justify
including it in the curated bundle. Use the separate guest-origin dataset for the
stronger country-based story.
