# Linztermine - Schlagworte

**Final verdict: USE WITH PREPARATION AS A HELPER ONLY.** Include it inside the
Linztermine bundle, not as a standalone hackathon dataset.

**Rainer verdict:** ⚠️ Borderline — a tiny, clean category taxonomy that is only
useful as a lookup/crosswalk, not as standalone data.

The live XML contains five top-level tags and about 25 subtags. A hand-authored
crosswalk can align festival categories such as Exhibition and Concert with city
event filters.

Convert XML to JSON and ship it with the crosswalk. It contains no events, dates,
or coordinates and should not appear independently in dataset discovery.
