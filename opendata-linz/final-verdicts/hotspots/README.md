# Öffentliche WLAN-Hotspots

**Final verdict: OPTIONAL.** Offer it as a clearly dated supporting layer, not as
current connectivity or crowd information.

**Rainer verdict:** ✅ Recommended — clean geo and usage data with a direct
spatial/name join to festival venues, with a 2022-vintage caveat.

About 134 WGS84 points include direct landmark matches such as AEC Dach,
Brucknerhaus, and Hauptplatz. Usage counts can support an illustrative digital-city
story.

Location and usage data are frozen in 2022; usage covers only January-July. Normalize
comma decimals and names, and never use the counts as a 2026 crowd or capacity
signal.
