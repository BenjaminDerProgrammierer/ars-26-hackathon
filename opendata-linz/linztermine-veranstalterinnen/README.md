## Beschreibung

Der Endpoint lieferte rund 1.611 Veranstalter mit ID, Name und optionalen Kontakt- oder Webangaben. Die Veranstalter-ID verbindet Einträge mit dem Linztermine-Eventfeed; Ars Electronica war bei der Prüfung als ID `7` vorhanden. Kontaktdaten können fehlen oder veraltet sein.
